# Estrutura do Projeto e Arquivos
@page estrutura_arquivos

## Arquivos do Projeto
O projeto está organizado nos seguintes componentes:

| Arquivo | Descrição |
| :--- | :--- |
| **`main-PL403.c`** | Contém a função principal (`main`) |
| **`functions-PL403.c`** | Implementação de todas as funções. |
| **`functions-PL403.h`** | Cabeçalho com as declarações de funções e estruturas. |
| **`makefile`** | Compilação |
| **`Doxyfile`** | Configuração do Doxygen. |
| **`mainpage.md`** | Página inicial da documentação. |
